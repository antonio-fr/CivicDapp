Initialisation du vote

Comment créer une élection avec Civic ?

1. Donner un nom à l'élection et l'organisateur.

2. Définir les différents choix (candidats, propositions,...)

3. Lister les votants.

4. Cliquer sur le bouton « Générer ».

5. Attendre l'initialisation du vote.

6. Télécharger le fichier vote-setup.json sur votre ordinateur.

Glisser / Déposer pour importer les fichiers de configuration JSON ici.
